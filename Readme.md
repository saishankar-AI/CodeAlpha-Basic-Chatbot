# Basic Rule-Based Chatbot

## Description

This project is a simple rule-based chatbot developed using Python.

The chatbot responds to predefined user inputs like greetings, questions, and exit commands.

## Features

- Greets the user
- Responds to "How are you?"
- Tells its name
- Ends conversation with "bye"

## Technologies Used

- Python

## Concepts Used

- Functions
- if-elif-else
- while loop
- input()
- print()

## How to Run

Open terminal and type:

python chatbot.py
